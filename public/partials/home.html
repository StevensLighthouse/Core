Creating a Stop

The first thing you'll want to do is to create a stop. Creating some stops will allow you to create your first tour.

In order to create a stop, you're going to need:

  • A name
  • A description
  • Either:
    • An Address
    • Latitutude and longitude coordinates
  • Some categories

After you create your stop, you can start adding photos to it from the stop edit page!

Creating a Tour

Adding a trip is just as easy as creating a stop!

Once you have some stops, you're going to want to start organizing them into tours. For that, you're going to need:

  • A name
  • A description
  • Some stops

While building a tour, you can drag your map to set the 'center' of your tour. When someone loads up that tour, they'll see it centered there.

Not able to finish your tour in time? You can always just save the tour and set the tour to not be visible yet -- this will mean that you and your editors will be able to keep working on the tour, but user's won't see it until you're ready.
